All About Raspberry Pi
The Mini Computer

What is Raspberry Pi?

Raspberry Pi is mini, single-board computer capable of doing almost everything that an ordinary computer can do. It is designed to help children learn the basics of computing and programming, and its small size makes it useful for lots of different projects. The Raspberry Pi looks very different from other computers as it comes without a case, so you can see its microchips and circuit board.




Raspberry Pi Line-Up

Have you ever seen a Raspberry Pi? There are lots of different models, and they do many amazing things. Some of the smaller models are great for putting inside other things, such as spy cameras. Other models can control robots, make games for you to play, and much more.

The Original Model B

This is an old Raspberry Pi model. It is still a perfectly good tool for learning how computers work, learning how to interface strange hardware, learning the Internet of Things. The telltale sign is that there are only two USB ports and 26 GPIO pins. If you find one of there, it will still work but it'll be slower than the newer model.

Pi 3

This is the latest version of the Model B and the best to use when you're getting started and learning how to program. The Pi 3 comes with Wi-Fi and Bluetooth. Two earlier versions - Model b+ and Pi 2 - look very similar to the Pi 3. Ask an adult to help you make sure you get the right model if you want a Pi 3.

Model A+

This almost-square Raspberry Pi model is great for projects that need a smaller computer. It's not as fast as the Pi 3 and doesn't have built-in Wi-Fi to connect it to the internet. It uses less power than the bigger models, which is useful if you need to use batteries to power it.

Pi Zero

This model is the cheapest and smallest of the Pi family. It needs adapters to be able to connect to a keyboard and mouse. It doesn't have GPIO pins, but you can solder them on. The Pi Zero is best suited to really small projects.

Tinkering

When you first learn to code, chances are you will write a program to display "Hello, World!" on the screen. When learning electronics, you will make something "light up" Before we build anything fancy, let's make your LED to light up!


Click each step as you complete it:


Step 1: Insert the LED into the breadboard

Choose an LED, and look at its legs carefully. Connect the long (positive) leg to the Pin 17 and then connect the short (negative) leg to the GND.

Step 2: Add a resistor

Place a resistor so it connects with the short (negative) leg of the LED.

Step 3: Wire to the Pi

Connect the wire to Pin 17 and another of wire to the GND pin (via a hole in the same breadboard column).

Step 4: Grounding

Finally, for the current to flow, it needs a ground. Connect another wire to the GND pin to complete the circuit.

Step 5: Code

You are going to program your LED in Python with the GPIO Zero library to control the LED.
